# ASCII Roguelike game

You are a daring explorer.

Your ultimate goal: to acquire the precious legendary artifact.

During one of your explorations, you fall into a hole.

You then find yourself in a chamber, lit by magical crystals, offering a warm atmosphere.

Could it be that you have laid your hands on the long‑coveted treasure…?

This is a simple text-based roguelike game using ASCII art for visuals. It's a fun little project inspired by classic roguelike mechanics.
As you move to different rooms,  you'll encounter various events, like meeting monsters, finding treasure chests or stumbling across a lost enigma to be solved.
